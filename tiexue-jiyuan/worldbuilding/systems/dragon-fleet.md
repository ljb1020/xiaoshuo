## Overview
2028年最顶尖的综合海上打击群，人类战争史上的巅峰暴力机器。拥有绝对的制空权、制海权和信息碾压能力。在1937年的旧世界里，这就是真神降临。

---

## 武器三级分类体系（核心防崩坏铁律）

### 🔴 A级：战略底牌，不可再生
用完即没，每次动用都必须"值"——这本身就是爽点（读者会为每一次A级武器的释放而尖叫）。
| 武器 | 代号 | 库存 | 用途 |
|------|------|------|------|
| 高超音速反舰导弹 | 鹰击-XX | 极少量 | 灭舰、斩首、定局 |
| 先进中距空空导弹 | 霹雳-XX | 有限 | 空战清场（开手即清零） |
| 精密制导重型反舰导弹 | 鹰击-YY | 有限 | 大舰队歼灭战 |
| 核潜艇重型线导鱼雷 | 鱼-XX | 极少量 | 深海绞杀 |
| 战术核弹头 | 东风-XX | 仅1-2枚 | 全书终极威慑（仅用一次） |

### 🟡 B级：有限维修，有限补充
可以在台湾基地进行简易维护和部分零件替换，但无法批量制造。
| 武器 | 用途 |
|------|------|
| 0VV大驱舰炮（130mm） | 舰队中近距火力压制 |
| 武直-10武装直升机 | 对地清剿、登陆支援 |
| 9VA主战坦克 | 装甲洪流碾压 |
| 步兵战车/装甲运输车 | 机械化突击 |
| 电子战/雷达系统 | 信息碾压（全图透视） |

### 🟢 C级：可工业降级复制（二代军核心装备）
利用带来的图纸和数控母机，在台湾兵工厂量产的"魔改版二战巅峰武器"。
| 武器 | 特点 |
|------|------|
| 改良半自动步枪（带红外瞄准） | 碾压三八大盖和M1 |
| RPG火箭筒 | 让步兵拥有反装甲能力 |
| 改进型迫击炮/重炮 | 超越二战平均火力 |
| 简化装甲车 | 薄皮但有机动力 |
| 初代雷达/无线电/夜视替代品 | 信息代差的降级延伸 |
| 合成氨炸药生产线 | 解决弹药自给 |

## Rules & Limitations
*   **绝对优势**：相控阵雷达全图透视、隐身战机超视距打击、信息化指挥链。对二战任何力量都是完全非对称的"降维打击"。
*   **核心限制**：A级弹药用完即没。B级装备战损后修复困难。必须在"无敌窗口期"内，攻取台湾建立工业基地，用C级量产武器武装二代军，实现从"一支舰队"到"一个国家"的转变。
*   **爽感逻辑**：限制不是为了让主角变弱，而是让每一次释放A级武器都变成全书的"名场面"。读者会为此尖叫。

## History
原系龙国执行台海封锁演习的绝对主力，因极小概率的地磁风暴被整体传送至1937年8月的东海。

## Practitioners
数千名受过极高现代教育、拥有坚定信仰和极强战术素养的现代军人（全员转化为"镇海卫"）。

## Impact on Society
不仅在物理上摧毁了樱花国的侵略野心，更在精神上粉碎了列强对东方的傲慢。他们是旧世界的掘墓人，新秩序的唯一缔造者。
