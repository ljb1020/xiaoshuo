## Description
原本被樱花国殖民统治的岛屿，在不到24小时内被镇海卫两栖突击舰和陆战队彻底扫平。这里拥有相对完备的基础农业和初步的工业框架，四周环海，是完美的天然避风港和种田基地。

## History
自1895年被割让后，经历了数十年的屈辱殖民统治。在1937年，镇海卫舰队以雷霆万钧之势空降台湾，将所有樱花国驻军和特务总督府彻底抹除，宣告回归华夏版图。

## Culture & Customs
当地百姓经历了极度的震惊到狂喜。镇海卫带来的现代纪律、现代医学和人人平等的现代理念，对当地社会造成了巨大的文明冲击和深度的文化认同。

## Notable Features
*   **左营/高雄军港**：被改造为镇海卫舰队的核心母港和核潜艇休整基地。
*   **新竹高新工业园区**：利用带来的现代设备和图纸，建立的超时空兵工厂，量产初级二战巅峰武器和燃油提炼厂。

## Current State
作为镇海卫的绝对大本营，全岛实行高度戒严的军管状态，所有樱花国人员被编入敢死劳改营。疯狂攀登科技树，吸收大量流亡内地的热血学子。
