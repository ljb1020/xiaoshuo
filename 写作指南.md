# 《铁血纪元：从东海升起的钢铁巨龙》小说创作指南

本指南旨在说明如何使用 `story-skills` (AI写作插件) 构建一部"降维打击"爽感 + "家国情怀"长篇军事架空小说。

## 📖 项目结构

```
xiaoshuo/                     ← workspace 根目录
├── .agents/                  ← 通用插件层（skill + workflow）
│   └── workflows/
│       └── auto-writer.md    ← 自动续写流水线（核心工作流）
├── 写作指南.md               ← 本文件
└── tiexue-jiyuan/            ← 小说项目根目录
    ├── story.md              ← 故事圣经（基调、规则、避讳词）
    ├── characters/           ← 角色卡（含 Voice Bank 和动态状态）
    ├── worldbuilding/        ← 世界观设定（战力、经济、地点）
    ├── plot/                 ← 大纲、时间线、伏笔库
    ├── chapters/             ← 正文章节
    └── scripts/              ← 自动化脚本（linter、timeline压缩等）
```

## 🚀 日常写作流程

**常规连载**只需一步：开新对话窗口 → 输入 `/auto-writer`。

AI 会自动完成：进度嗅探 → 加载设定 → 生成细纲 → 写正文 → Linter校验 → 字数防呆 → 更新时间线和角色状态。

详见 `.agents/workflows/auto-writer.md`。

## 🛡️ 防和谐代称（核心基准）

所有代称规则定义在 `story.md` Notes 部分，并由 `scripts/linter.py` 自动替换。

## 📐 建设性修改

每次让 AI 写完章节后，您可以直接在生成的 Markdown 文件中手动修改。AI 在写下一章时，永远以当前文件系统中的最新内容为基准。
