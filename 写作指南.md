# 《降临1937：现代舰队横跨东海》小说创作指南

本指南旨在说明写作平台的结构和使用方式。

## 📖 两层架构

```
xiaoshuo/                     ← $ROOT — 写作平台（所有小说共享）
├── .agents/                  ← 插件层（skill + workflow）
├── scripts/                  ← 平台级脚本（linter、wordcount、compress_timeline）
├── BACKLOG.md                ← 跨窗口持久化待办清单
├── 写作指南.md               ← 本文件
├── GEMINI.md                 ← Agent 行为约束
└── tiexue-jiyuan/            ← $NOVEL — 第一本小说
    ├── story.md              ← 故事圣经（基调、规则、避讳词、去AI味、修辞限速）
    ├── characters/           ← 角色卡（含 Voice Bank 和动态状态）
    ├── worldbuilding/        ← 世界观设定（战力、经济、地点）
    ├── plot/                 ← 大纲、时间线、伏笔库
    └── chapters/             ← 正文章节
```

后续新增小说项目时，在 `$ROOT` 下新建平行目录即可，共享 `.agents/` 和 `scripts/`。

## 🚀 日常写作流程

开新对话窗口 → 输入 `/auto-writer` → AI 自动完成全流程。

详见 `.agents/workflows/auto-writer.md`。

## 📐 手动修改

每次 AI 写完章节后，可以直接在 Markdown 文件中手动修改。AI 下一章永远以当前文件系统中的最新内容为基准。
